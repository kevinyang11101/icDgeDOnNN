# 图书管理系统简介 java322

## 项目概述

本图书管理系统是基于SSM（Spring + SpringMvc + Mybatis）与Vue的前后端分离项目。采用MVC三层体系结构，利用现代化技术手段，实现了Web系统的便捷性与高效性。

## 技术栈

- 前端：Vue
- 后端：Spring + SpringMvc + Mybatis
- 构建工具：Maven
- 数据库：MySQL

## 功能模块

- 用户管理
- 图书管理
- 借阅管理
- 系统管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- JDK版本：1.8及以上
- 数据库版本：MySQL 5.7及以上
- 操作系统：Windows/Linux/MacOS

## 部署步骤

1. 准备环境：安装JDK、MySQL、Maven等
2. 导入数据库：执行数据库脚本，创建项目所需表结构
3. 部署后端：通过Maven构建项目，并运行主程序
4. 部署前端：将前端代码部署至Web服务器

## 目录结构

```
book-management-system
```

## 核心亮点

- 基于B/S结构，支持多平台访问
- 采用前后端分离，方便开发与维护
- 利用SSM框架，确保系统的高效与稳定
